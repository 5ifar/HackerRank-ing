
### Q5. [Loops](https://www.hackerrank.com/challenges/python-loops/problem?isFullScreen=true)
Code:
```python
i = 0
while i < n:
    print(i ** 2)
    i = i+1
```
Optimized Code:
```python
i = range(n)
for item in i:
    print(item ** 2)
```
